Azure OpenAI

How to get access

  1. Create your service on the Core Delivery Platform
  2. Request LLM API access through the CDP portal
  3. Integrate the APIs into your application
  4. Deploy through the CDP pipeline

Important: LLM access for production applications must be requested through the CDP portal. This ensures proper governance and monitoring for live services.
